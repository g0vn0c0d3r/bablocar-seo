## 3. Модификаторы

Модификаторы взяты из `seo_strategy/programmatic_geo_strategy.md`.

Статусы предварительные. Их еще согласуем перед генерацией.

| Модификатор | Slug | Предварительная оценка | Почему |
| --- | --- | --- | --- |
| онлайн | `online` | Берем | Подходит к продукту, усиливает онлайн-сценарий, безопасно при аккуратной формулировке. |
| на карту | `na-kartu` | Берем | Понятный способ получения денег, можно раскрывать как сценарий заявки / выплаты без жестких обещаний. |
| без визита | `bez-vizita` | Берем | Хорошо ложится на дистанционный сценарий и отличается от базовой страницы. |
| без справок | `bez-spravok` | Берем | Можно делать как гео-модификатор, если пишем мягко: справки могут не потребоваться, но условия зависят от партнера. |
| без подтверждения дохода | `bez-podtverzhdeniya-dohoda` | Берем | Близко к `без справок`, но формулировка может собирать отдельный хвост и важна для пользователей без стандартного подтверждения дохода. |
| с плохой кредитной историей | `s-plohoi-ki` | Берем | Важный проблемный сценарий, но нельзя обещать одобрение. |
| с просрочками | `s-prosrochkami` | Берем | Отдельный проблемный сценарий, можно вести трафик на брокерскую страницу с осторожными формулировками. |
| без кредитной истории | `bez-kreditnoi-istorii` | Берем | Отличается от плохой КИ: пользователь может не иметь кредитной истории, а не иметь негативную. |
| без стоянки | `bez-stoyanki` | Берем | Важный ПТС-сценарий: пользователь хочет оставить авто у себя. Нужна проверка условий партнеров. |
| без изъятия авто | `bez-izyatiya-avto` | Берем | Возможная низкочастотная посадочная под страх, что автомобиль заберут. Формулировка отличается от `без стоянки`. |
| с правом пользования | `s-pravom-polzovaniya` | Берем | Отдельная формулировка вокруг права пользоваться автомобилем после оформления, можно забрать как НЧ-модификатор. |
| срочно | `srochno` | Берем | Можно говорить про быструю онлайн-заявку, предварительное решение и оперативную обработку без обещания гарантированной выдачи. |
| быстро | `bystro` | Берем | Скоростной сценарий близок к `срочно`, но может собирать отдельный хвост. В тексте держим безопасные формулировки. |
| для бизнеса | `dlya-biznesa` | Берем | Отдельный сегмент с другим контекстом заявки и мотивацией пользователя. |
| для юрлиц | `dlya-yurlic` | Берем | B2B-сценарий, требует отдельного текста и условий. |
| для ИП | `dlya-ip` | Берем | Бизнес-сегмент, отличается от базового физлица-сценария. |
| для физлиц | `dlya-fizlic` | Берем | Фиксируем как отдельный сегмент по решению, чтобы явно покрыть формулировки для физических лиц. |
| для самозанятых | `dlya-samozanyatyh` | Берем | Отдельный сегмент с особенностями подтверждения дохода и занятости. |
| без отказа | `bez-otkaza` | Берем | Может дать трафик; как брокерская страница должна говорить про подбор и индивидуальное рассмотрение, без обещания гарантированного одобрения. |
| авто остается у клиента | `avto-ostaetsya-u-klienta` | Не берем отдельно | Скорее формулировка для текста / FAQ внутри страниц `bez-stoyanki` и `bez-izyatiya-avto`, а не отдельный URL. |
| рефинансирование | `refinansirovanie` | Отдельный интент | Это не просто модификатор к займу под ПТС, а отдельный сценарий перезалога / закрытия старого обязательства. |
| перезалог | `perezalog` | Отдельный интент | Близко к рефинансированию; лучше выносить в отдельный интент, а не множить как модификатор. |

## 4. Получающаяся схема

Базовая схема для каждого города:

```text
/{city}/
/{city}/zaim-pod-pts/
/{city}/zaim-pod-pts/{modifier}/
```
