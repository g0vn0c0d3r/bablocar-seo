# Проверка meta и hero для pSEO-страниц

Актуальные критерии проверки опубликованных страниц BabloCar.

## Что проверяем

| Блок | Поля |
| --- | --- |
| Meta | `title`, `description` |
| Hero | бровь, H1, lead |

## Критерии дублей

| Проверка | Косяк | Норма |
| --- | --- | --- |
| `title` | одинаковый 1-в-1 на разных страницах | отличается хотя бы городом, интентом или модификатором |
| `description` | одинаковый 1-в-1 на разных страницах | отличается под задачу страницы |
| Hero H1 | одинаковый 1-в-1 на разных страницах | содержит нужную подстановку города, интента или модификатора |
| Hero lead | одинаковый 1-в-1 на разных типах страниц | может быть шаблонным, если меняются подстановки |

Проценты пересечения не считаем. Если текст не одинаковый дословно, это не дубль.

## Норма по hero-шаблонам

| Тип страницы | Бровь | H1 | Lead |
| --- | --- | --- | --- |
| Городской хаб | `{city}` | широкий городской интент + `{city}` | общий городской сценарий |
| Интент | `{category}` | `{category}` + `{city}` | заявка по `{category}` в `{city}` |
| Модификатор | `{tag}` | `{category}` + `{tag}` + `{city}` | заявка по `{category}` + `{tag}` в `{city}` |

Шаблонность в hero допустима и ожидаема. Важно, чтобы реальные H1 не дублировались 1-в-1.

## Норма по meta

| Тип страницы | Что должно быть в `title` и `description` |
| --- | --- |
| Городской хаб | город + широкий интент про ПТС/авто |
| Интент | город + категория |
| Модификатор | город + категория + условие |

Meta не обязаны повторять hero. Они могут быть менее шаблонными, если попадают в запрос пользователя.

## Короткий итог последней проверки

| Проверка | Статус |
| --- | --- |
| Hero-шаблоны | ок |
| Уникальность H1 | ок |
| Meta попадает в запрос | ок |
| Точные дубли `title` | нет |
| Точные дубли `description` | нет |
