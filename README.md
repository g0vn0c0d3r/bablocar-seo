# bablocar-seo

Рабочий Git-репозиторий для SEO-документации проекта bablocar.

Сейчас проект начинается с чистой дорожной карты. Старые файлы, семантика, гипотезы и структура не используются автоматически, пока пользователь заново не подтвердит их в рамках нового процесса.

## Роль README

`README.md` — главная база знаний проекта на верхнем уровне. В нем фиксируем:

- карту документов и папок проекта;
- текущий статус работ;
- глобальные принципы SEO-стратегии;
- выбранную URL-логику, интенты и `modifier`-подход;
- принципы структуры лендингов и покрытия семантики в контентных блоках;
- важные решения, которые нельзя потерять между этапами.

Детальные таблицы, аудиты, брифы, гипотезы и ТЗ живут в отдельных файлах. README должен оставаться коротким навигатором и стратегическим слоем, без перегруза техническими деталями.

### Принцип SEO-структуры

Один из базовых принципов будущей стратегии — строить страницы как управляемую матрицу:

```text
/{geo}
/{geo}/{core_intent}
/{geo}/{core_intent}/{modifier}
```

Термины:

- `geo` — город или региональная посадочная, например `/samara`;
- `core_intent` — базовый коммерческий интент, например `/samara/zaim-pod-pts`;
- `modifier` — уточнение интента, например `/samara/zaim-pod-pts/bez-otkaza`.

В админке такие уточнения могут называться тегами, но в SEO-документах рабочий термин — `modifier`, потому что это не произвольный тег, а управляемый хвост интента.

Семантику нужно покрывать не механическим списком ключей, а структурой лендинга:

- H1, Title и Description закрывают основной `geo + core_intent`;
- первый экран, калькулятор и CTA закрывают коммерческий сценарий;
- блоки условий, документов, требований и FAQ закрывают уточняющие хвосты;
- рискованные формулировки раскрываются безопасно: без гарантий, абсолютов и обещаний 100% одобрения;
- внутренняя перелинковка связывает город, core-intent страницы и modifier-страницы.

Не каждый `modifier` автоматически умножается на каждый город и каждый `core_intent`. Страница создается только там, где есть спрос и можно сделать содержательно отличимую посадочную.
