# bablocar-seo

Рабочий Git-репозиторий для SEO-документации проекта bablocar.

Сейчас проект начинается с чистой дорожной карты. Старые файлы, семантика, гипотезы и структура не используются автоматически, пока пользователь заново не подтвердит их в рамках нового процесса.

## Документы

- `roadmap.md` — дорожная карта SEO-проекта.

## Правило

В Git храним управляемые документы и код. Сырые выгрузки, Excel, PDF и тяжелые экспорты не коммитим без отдельного решения.
