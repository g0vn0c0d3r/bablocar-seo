# bablocar-seo

Главная база знаний SEO-проекта Bablocar.

README нужен как короткая карта решений: где мы находимся, по какой логике строим SEO, какие документы за что отвечают и что делать дальше. Подробная методология лежит в `seo_strategy/programmatic_geo_strategy.md`.

## Статус

Закрыты базовые этапы:

- продуктовая база;
- партнерские условия и безопасные формулировки;
- конкурентный блок;
- сбор и очистка семантики;
- базовая методология programmatic geo SEO.

Текущий рабочий этап: `Структура сайта` + `Шаблоны страниц` + аудит первых коммерческих лендингов.

Ближайшая задача: довести структуру городского хаба, гео-интента и гео-модификатора до согласованного шаблона, затем масштабировать через города, интенты и модификаторы.

## Выжимка стратегии

Сначала структура, потом пояснения к уровням.

```text
Уровень 1
Главная
/
│
├── Уровень 2: /samara/
│   Городской хаб Самары
│   │
│   ├── Уровень 3: /samara/zaim-pod-pts/
│   │   Займ под ПТС в Самаре
│   │   │
│   │   └── Уровень 4: гео-модификаторы
│   │       ├── /samara/zaim-pod-pts/bez-spravok/
│   │       ├── /samara/zaim-pod-pts/s-plohoi-ki/
│   │       └── /samara/zaim-pod-pts/bez-stoyanki/
│   │
│   ├── Уровень 3: /samara/kredit-pod-pts/
│   │   Кредит под ПТС в Самаре
│   │
│   └── Уровень 3: /samara/dengi-pod-pts/
│       Деньги под ПТС в Самаре
│
├── Уровень 2: /kazan/
│   Городской хаб Казани
│   │
│   ├── Уровень 3: /kazan/zaim-pod-pts/
│   ├── Уровень 3: /kazan/kredit-pod-pts/
│   └── Уровень 3: /kazan/dengi-pod-pts/
│
└── Уровень 2: /moskva/
    Городской хаб Москвы
    │
    ├── Уровень 3: /moskva/zaim-pod-pts/
    ├── Уровень 3: /moskva/kredit-pod-pts/
    └── Уровень 3: /moskva/dengi-pod-pts/
```

Основная модель programmatic SEO:

```text
города x коммерческие интенты x разрешенные модификаторы
```

URL-модель:

```text
/{город}/
/{город}/{интент}/
/{город}/{интент}/{модификатор}/
```

Рабочая логика:

```text
очищенная семантика
-> коммерческие интенты
-> алиасы интентов
-> модификаторы
-> правила применимости
-> городские хабы
-> гео-страницы интентов
-> гео-модификаторы
-> статьи / FAQ
```

Короткие правила из драфта стратегии:

- займ, кредит и деньги - разные коммерческие интенты;
- ПТС, авто, машина, грузовик, спецтехника, мотоцикл - объекты залога;
- авто и автомобиль объединяем в объект "авто";
- машину держим отдельным объектом;
- алиасы не становятся URL;
- модификаторы собираем один раз из всей семантики;
- модификаторы применяются к интентам по правилам;
- на старте каркас сайта - гео-сетка;
- гео-страницы - основная масса programmatic SEO;
- к федеральным хабам можно вернуться позже.

Пример URL на интенте `займ под ПТС`:

```text
/samara/
/samara/zaim-pod-pts/
/samara/zaim-pod-pts/bez-spravok/
```

## Роли документов

- `roadmap.md` - порядок этапов и текущий статус проекта.
- `product/product_brief.md` - продуктовые факты, ограничения и позиционирование.
- `partners/partner_conditions.md` - агрегированные условия партнеров.
- `partners/safe_claims.md` - безопасные и рискованные формулировки.
- `competitors/findings.md` - выводы по рынку и конкурентным подходам.
- `seo_strategy/programmatic_geo_strategy.md` - главный стратегический документ: сущности, URL-модель, схема сайта, правила применимости и будущие справочники.
- `site_structure/zaim-pod-pts.md` - первая карта сайта по выбранному интенту.
- `site_structure/commercial_landing_template.md` - базовый шаблон коммерческого лендинга для городских хабов, гео-интентов и гео-модификаторов.
- `site_structure/page_generation_rules.md` - правила масштабирования: город -> интент -> модификатор.
- `site_structure/commercial_landing_audit.md` - короткие аудиты готовых коммерческих страниц без скриншотов.
- `site_structure/commercial_landing_block_review.md` - подробный разбор блоков с визуальными референсами.
- `site_structure/service_pages.md` - список служебных страниц перед запуском.
- `site_structure/service_pages_drafts/` - черновики служебных страниц.
